# CloserDojo — Design System Implementation

CloserDojo is a role-play call training tool and SaaS product in development. It uses the Autumn Design System with an Amber primary color to convey energy and confidence.

## Product-specific patterns

- Role-play call interface: full-screen, minimal chrome — content is king
- Scoring cards: use `success`/`warning`/`danger` variants
- Leaderboard: data-dense, use `micro` typography size
- Call recordings: audio player inline in cards
