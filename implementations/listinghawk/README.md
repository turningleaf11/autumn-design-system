# ListingHawk — Design System Implementation

ListingHawk is an AI deal sourcing tool that scans markets and surfaces qualifying listings into a GHL pipeline. Uses Teal primary color for a sharp, data-driven feel.

## Product-specific patterns

- Deal cards: show key numbers prominently (price, ARV, equity %)
- Market badge: color-coded by market/city
- Pipeline stages: horizontal flow, not vertical list
- Scan results: table-dense layout with sortable columns
